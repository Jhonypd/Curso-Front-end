O que é um Front - End?

1- Construção de interfaces. Em programação, front end diz respeito a tudo que é visível pelo usuário, como os menus, interfaces e outros elementos gráficos.

2- Implementação do design. É importante esclarecer que o trabalho desse desenvolvedor não é o mesmo de um designer. Enquanto o designer elabora o esquema de cores e pensa na experiência do usuário, o Desenvolvedor Front End fará o trabalho de implementação desse design.

3- Corrigir erros visuais. O Desenvolvedor Front End também tem a tarefa de identificar e corrigir esses erros ao longo do desenvolvimento.
